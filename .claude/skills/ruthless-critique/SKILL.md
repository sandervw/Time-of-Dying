---
name: ruthless-critique
description: Deliver 5-8 sharp criticisms of a provided fiction document, one at a time, with a suggested fix for each, editing the document after each response. Use when the user asks for a ruthless critique, hard feedback, or mentions "ruthless-critique".
---

Critique the provided fiction document ruthlessly. Start by confirming which document you're critiquing if it isn't obvious.

Identify 5-8 sharp criticisms. Look for:
- Overused fiction tropes and cliches
- Confusing or contradictory plot elements
- Generic, non-specific "details" that could belong to any story
- Telegraphed twists, predictable beats

Do NOT critique grammar, punctuation, or line-level prose mechanics.

Present criticisms one at a time. For each:
- Name the problem bluntly. Quote or cite the offending passage.
- Explain why it weakens the work.
- Propose a concrete fix.

After the user responds, edit the document to apply the agreed fix before moving to the next criticism. If the user rejects, leave the document alone and move on.

Keep each criticism brief.
