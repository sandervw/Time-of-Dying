# Time of Dying

**Time of Dying** is an AI-assisted fiction development pipeline built with Claude Code, the Anthropic API/SDK, and custom prompt engineering skills/agents. It uses structured prompt workflows — including folklore generation, literary style revision, character analysis, and scene writing — to collaboratively develop and maintain internal consistency across a large-scale speculative fiction setting. The repository contains both the tooling (skills, agents, API scripts) and the setting artifacts they produce.

## What's in here?

This repository contains artifacts for creating stories - and refining the setting itself - of the Time of Dying fiction universe. This setting is built collaboratively with claude/claude-code.

The repository consists of:
1. **`.claude/`** - Agents and skills used in story/setting creation (character analysis, folklore generation, literary revision, etc.)
2. **`Input/`** - Context files (markdown/JSON) fed to the Claude API
3. **`Javascript/`** - Scripts for calling the Claude API/SDK (`main.js` entry point, module wrappers)
4. **`Output/`** - Default destination for files generated by skills and agents
5. **`Setting/`** - Setting documents, organized hierarchically:
   - `Time-of-Dying.md` - Setting overview for the Time-of-Dying project; a 400-word summary of the info.
   - `Folklore.md` - Random pieces of folklore for the setting
   - `Ideas.md` - Running scratchpad for rough ideas and notes (temporary; items get incorporated or discarded)
   - `Mythology.md` - Main setting summary mapping real-world objects to classical myth categories
   - `Folklore/` - Deep-dive docs for random pieces of cofirmed setting lore
   - `Mythology/` - Deep-dive docs for each defined mythology element
   - `Potential-Folklore/` - Generated folklore not yet incorporated or confirmed to be true to the setting
6. **`Stories/`** - In-progress narratives and scene outlines

Almost all files are:
- .json (collections of elements: IE. 'monsters.json')
- markdown (setting/story documents)
- javascript
