# Source Imprint — Dreamreel

**Sentence length:** Short declaratives (4-8 words) alternate with mid-length action chains (20-35 words); no very long periodic sentences.

**Clause stacking:** Right-branching; action appended by coordination ("and," "then"); minimal subordination.

**Register:** Plain, slightly archaic diction; no contractions; concrete nouns dominate.

**Sensory:** Visual and kinetic primary; interior states named bluntly ("startled," "frenzy"), not expanded.

**Tics:** Repeated "and" chains inside action; sparse but exact colour/clothing detail mid-action.

---

**Do:** Short declaratives to land blows; chain action with "and"; name emotion in one word.
**Don't:** Stack subordinate clauses; linger on interiority; use Latinate abstractions.
